# Install and use the Mask

An SVG mask that cover all the window contents except the one specified by certain position and sizes values

## Install [@dezignerblake/mask](https://www.npmjs.com/package/@dezignerblake/mask)

```sh npm2yarn
npm i @dezignerblake/mask
```

## Usage

```js
import { Mask } from '@dezignerblake/mask'

function App() {
  const sizes = {
    width: 100,
    height: 100,
    top: 100,
    left: 100,
  }

  return (
    <>
      {/* ... */}
      <Mask sizes={sizes} />
    </>
  )
}
```
