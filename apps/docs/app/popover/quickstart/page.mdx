# Install and use the Popover

A popover positioned based on certain values

## Install [@dezignerblake/popover](https://www.npmjs.com/package/@dezignerblake/popover)

```sh npm2yarn
npm i @dezignerblake/popover
```

## Usage

```js
import { Popover } from '@dezignerblake/popover'

function App() {
  const sizes = {
    bottom: 0,
    left: 0,
  }

  return (
    <>
      {/* ... */}
      <Popover sizes={sizes}>
    </>
  )
}
```
